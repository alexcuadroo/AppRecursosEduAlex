# AppRecursosEduAlex

Aplicación portable para listar y visualizar recursos educativos de **EduAlex**. Consulta una API REST y muestra materiales filtrables por materia, año, tipo y formato.

## Características

- Explorar todos los recursos con paginación
- Filtros por materia, año, tipo y formato
- Búsqueda por texto
- Pestañas: **Todos**, **Últimos** y **Más visitados**
- Vista previa inline de recursos
- Detección automática de actualizaciones

## Portable — sin instalación

1. Descargar el `.zip` de la [última release](https://github.com/alexcuadroo/AppRecursosEduAlex/releases/latest)
2. Descomprimir en cualquier carpeta
3. Ejecutar `AppRecursosEduAlex.exe`
4. (Opcional) Anclar a la barra de tareas para acceso rápido
